# WED-PART-2

Thandi’s Bakery Website
Project Overview

Thandi’s Bakery is a simple bakery website created using HTML and CSS.
The website allows customers to:

View bakery products
See prices and product information
Make enquiries and orders
Contact the bakery
Access customer service support
Pages Included
1. Home Page

The landing page introducing Thandi’s Bakery and its homemade baked goods.

Features
Bakery introduction
Navigation menu
Modern bakery background
Responsive layout
2. About Page

Provides information about the bakery.It also provies clear goals for the company.

Features
Bakery history
Mission and vision
Homemade baking information
3. Products Page

Displays all bakery products available for customers.

Products Included
Scones
Biscuits
Banana Bread
Buns
Muffins
Addits (extras)
Features
Product images
Product descriptions
Product prices
Styled product cards
Responsive image sizing
4. Enquiry / Customer Service Page

Allows customers to contact the bakery.

Features
Customer complaint form
Order support
Product enquiry section
Message box for customer feedback
Information Collected
Full Name
Email Address
Phone Number
Product selection
Customer message
5. Contacts Page

Contains bakery contact details and location.

Features
Social media links
Embedded Google Map
Contact information
Technologies Used
HTML5
CSS3
Styling Features

The website includes:

Glassmorphism design
Responsive product cards
Flexbox layouts
Hover animations
Modern navigation bar
Styled forms
Centered social media icons
